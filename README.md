# Zoo de Babentruk

Application web de gestion d’un zoo développée en **PHP** avec une **base de données Oracle**.  
Ce projet permet de gérer les animaux, les espèces, les enclos, le personnel, les soins, les nourrissages, les parrainages, les boutiques et les réparations.

## Fonctionnalités

- Authentification des utilisateurs
- Gestion des rôles et des permissions
- Gestion des animaux
- Gestion des espèces
- Gestion des enclos et des zones
- Gestion du personnel
- Gestion des soins et des nourrissages
- Gestion des parrainages
- Gestion des boutiques
- Gestion des réparations
- Tableau de bord avec statistiques

## Installation

### 1. Cloner le projet

```bash
git clone https://github.com/aminegitcode/zooland.git
cd zooland
```
### 2. Configurer le serveur web

Placer le projet dans le dossier du serveur web Apache.

### 3. Initialiser la base de données

Exécuter le fichier `bd.sql` afin de créer la structure de la base de données et insérer les données nécessaires au fonctionnement du projet.

### 4. Configurer la connexion à la base de données

Modifier le fichier `myparam.php` avec les paramètres de connexion à la base de données Oracle.

### 5. Lancer le projet

Démarrer Apache puis ouvrir le projet dans le navigateur.

## Comptes de démonstration

Selon la configuration du projet, des comptes de démonstration peuvent être utilisés :

- `admin` / `admin123`
